## Project 2 - Chess (CodersCamp)

### How to run

Clone this repo

```
git clone <repo>
```

Install all dependencies

```
npm install
```

### Run webpack-dev-server

```
npm start
```

### Local host address:

http://localhost:8080/

### Work rules

- don't push directly on master branch
- create your own branch
- after commiting changes, push your branch and make a pull request
